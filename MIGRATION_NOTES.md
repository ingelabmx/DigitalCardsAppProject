Esta fase agrega contexto documental y un esqueleto ASP.NET Core paralelo. No reemplaza todavia la aplicacion Web Forms. MySQL HostGator, Google Wallet real, SMTP real, Apple Wallet `.pkpass` y Apple Wallet updates por Web Service/APNs quedan disponibles por configuracion local controlada; los fakes siguen siendo el default para desarrollo, CI y Playwright. El flujo moderno de negocio ahora usa cookie auth, ya no depende de `businessId` visible en URL/formularios, migra passwords de negocio a hashes modernos sin modificar la columna legacy, empieza la operacion diaria desde `/Business/Cards`, usa links Wallet opacos para correos nuevos, registra auditoria de sellos en `StampLedger`, permite que el admin legacy habilite negocios piloto desde `/Admin/Businesses`, registra negocios nuevos desde `/Admin/CreateBusiness` con compatibilidad legacy, administra datos/reset de contrasena desde `/Admin/BusinessProfile/{businessId}`, permite gestionar admins desde `/Admin/AdminUsers` sin registro publico, retira la allowlist de clientes del flujo operativo para que el piloto se controle solo por negocio, agrega login/dashboard moderno de cliente con cookie `DigitalCards.Client`, migra passwords de cliente a `ModernClientCredential` con cambio de contrasena moderno, mejora el dashboard cliente con perfil/resumen/estados Wallet y paridad visual legacy, convierte `/Business/Dashboard` en resumen operativo con tarjetas recientes y StampLedger, agrega branding basico de negocio para correos, Wallet y dashboard cliente, centraliza plantillas de correo para Wallet, bienvenida, reset y alertas internas, agrega reset password por email con tokens opacos one-time para clientes y negocios, agrega `/Admin/Support` como diagnostico seguro exportable de clientes, negocios, tarjetas, Wallets, StampLedger y LegacyWalletSync, documenta la operacion estable de `app.puntelio.com` con scripts de arranque/health en `ops/windows`, define el reemplazo gradual de Web Forms por negocio con gates y rollback, agrega estado formal de activacion por negocio para distinguir `LegacyOnly`, `PilotModern`, `ModernPrimary` y `LegacyRetired`, permite subir logos publicos por negocio para UI, correo, Google Wallet y Apple Wallet cuando el asset es PNG, documenta el inventario visual Web Forms para guiar la paridad de UI en Razor Pages, agrega un shell autenticado con sidebar/header/footer estilo Web Forms, y ajusta las pantallas admin a una presentacion mas cercana al dashboard legacy.
